# Portal Chatbot for University of Cape Coast

Welcome to the Portal Chatbot, an AI-powered chatbot designed to answer questions and provide information about the University of Cape Coast. This tool is particularly helpful for students, faculty, and visitors to quickly find answers regarding university details. This chatbot is built using a responsive AI framework compatible with Python 3 and above.

## Features

- Provides information about admission processes, programs, and facilities at UCC
- Utilizes Natural Language Processing for understanding user queries
- Offers quick and accurate responses to enhance user experience

## Prerequisites

Before you set up your Portal Chatbot, make sure you have:
- Python 3.6 or newer

## Installation

Install Portal Chatbot with the following steps:

1. Clone the repository:

```bash
git clone https://github.com/your-github/portal-chatbot.git
```

2. Navigate to the project directory:

```bash
cd portal-chatbot
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

The `requirements.txt` file should list all Python libraries that your project depends on, and they will be installed using:

```bash
pip install -r requirements.txt
```

## Usage

Run Portal Chatbot using the following command:

```bash
python3 main.py
```
